Implementace

Backend

Celý backend infomačného systému je postavený na webovom mikroframeworku Flask.
Databázový model je vytvorený pomocou objektovo orientovaného nástroja SQLAlchemy,
ktorý sa používa za pomoci jazyka Python.
Dáta v databáze boli vygenerované pomocou knižnice Faker.

Frontend

Základ tvorí šablónovací systém, ktorý je používaný spolu s Flaskom, tj. Jinja2.
Je vytvorený pre jazyk Python a vychádza zo syntaxe Django. Umožňuje priamu komunikáciu s backendom.
Samozrejmosťou je použitie HTML5 pre statické časti.
Základom vizualizácie je webový framework Bootstrap.
Pre dynamické časti je použitý jazyk JavaScript spolu s knihovňou JQuery.
V niektorých častiach je použitý AJAX pre asynchronné získanie dát.

Instalace

  • Softwarové požadavky: Python >= 3.3, PostgreSQL 9.2/SQLite 3.13.0, npm 3.10.9
  • Inštalácia Python balíkov: pip3 install -r requirements.txt
  • Inštalácia JavaScript balíkov: npm install
  • Zbundlovanie JavaScriptu: webpack
  • Jak rozbalit, konfigurovat, inicializovat databází, ...
