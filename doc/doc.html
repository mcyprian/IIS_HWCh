Implementace

Zadanie projektu

Pri návrhu a následnej implementácií systému sme sa držali zadania a jeho upresnenia
z predmetu IDS. Snažili sme sa pokryť všetky prípady použitia a poskytovať všetky informácie
tak ako sú v zadaní špecifikované.
Výsledný systém sa v niektorých aspektoch líši od diagramu prípadov použitia z predmetu IDS
a to hlavne z dôvodu že prípady použitia v predchádzajúcom projekte neboli narhnuté optimálne.
Ako príklad môžme uviesť úpravy štatistiky hráča, ktoré priamo vyplývajú zo zadaných
udalostí zápasov, alebo zmenu stavu zápasu na "odohratý", táto informácia by bola redundantná,
pretože ju je možné odvodiť z prítomnosti formácií alebo udalostí naviazaných na konkrétny zápas.

Backend

Celý backend infomačného systému je postavený na webovom mikroframeworku Flask.
Pri implementácií typických súčastí webových aplikácií ako napríklad autorizácia, založená na roliach uživateľov sú využité rozšírenia tohoto frameworku.
Databázový model je vytvorený pomocou nástroja SQLAlchemy,
ktorý umožňuje mapovanie tried jazyka Python na tabuľky databázy.
Ukážkové mená, dátumy a čísla v databáze boli vygenerované pomocou knižnice Faker.

Frontend

Základ tvorí šablónovací systém Jinja2, ktorý je používaný spolu s Flaskom.
Bol vytvorený pre jazyk Python a vychádza zo syntaxe Django. Umožňuje priamu komunikáciu s backendom.
Samozrejmosťou je použitie HTML5 pre statické časti.
Základom vizualizácie je webový framework Bootstrap.
Pre dynamické časti je použitý jazyk JavaScript spolu s knihovňou JQuery.
V niektorých častiach je použitý AJAX pre asynchronné získanie dát.

Instalace

  • Softwarové požadavky: Python >= 3.3, PostgreSQL 9.2/SQLite 3.13.0, npm 3.10.9
  • Rozbalenie archívu: unzip xbaric01.zip; cd IIS_HWCh
  • Inštalácia Python balíkov: pip3 install -r requirements.txt
  • Inštalácia JavaScript balíkov: npm install
  • Zbundlovanie JavaScriptu: webpack
  • Inicializácia databázy a naplnenie ukážkovými dátami: python3 manage.py init_db
  • Lokálne spustenie aplikácie na porte 5000: python3 manage.py runserver
